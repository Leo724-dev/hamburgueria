<h1>🍔 DevBurger - Cardápio Digital Dinâmico</h1>

Este projeto é uma plataforma interativa de cardápio digital voltada para uma hamburgueria moderna. A aplicação permite que o usuário visualize os itens disponíveis, aplique descontos promocionais, filtre categorias específicas e calcule o valor total dos produtos de forma totalmente dinâmica.

<h2>📝 Sobre o Projeto</h2>

O foco principal é oferecer uma experiência de menu digital fluida e automatizada, simulando funções essenciais de um e-commerce de alimentação:
- **Exibição Dinâmica:** Renderização estruturada de produtos (como X-Salada, X-Bacon, Monstruoso, opções veganas, entre outros) manipulando arrays de objetos.
- **Mapeamento de Descontos:** Aplicação automatizada de porcentagens de desconto sobre os preços originais ao acionar a respectiva opção no menu superior.
- **Redução e Somatória:** Lógica para somar o valor total de todos os itens exibidos na tela em tempo real.
- **Filtros Condicionais:** Filtragem ágil para exibir apenas categorias específicas de produtos (como lanches tradicionais ou opções exclusivamente veganas).

<h2>🛠️ Tecnologias Utilizadas</h2>

HTML5: Estruturação semântica do grid de produtos e botões de controle.

CSS3: Estilização avançada utilizando imagens de fundo integradas, bordas iluminadas nos cards, tipografia estilizada e layout em Grid/Flexbox totalmente alinhado.

JavaScript (ES6+): Uso extensivo de métodos de array de alta ordem (como `map()` para descontos, `filter()` para os lanches veganos, `reduce()` para a somatória total e `forEach()` para a renderização dinamicamente no DOM).

<h2>✨ Funcionalidades</h2>

[x] Mapeamento de Preços: Aplicação de descontos globais em tempo real em todos os produtos do cardápio.

[x] Filtro Vegano: Separação automática de lanches vegetarianos/veganos com apenas um clique.

[x] Somatória de Valores: Cálculo matemático instantâneo de todos os itens visíveis na tela.

[x] Responsividade: Layout adaptável e scannável para telas de smartphones, tablets e desktops.
